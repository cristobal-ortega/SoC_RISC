Features:  
- 2 Stages:
  - Fetch  
  - Execution  
- Instructions:  
	- Arithmetic operations  
	- Load/Store (word/byte)  
	- Comparisons
	- Sign extensions  
	- Immediate operations  
	- Input/Output
	- Jumps
- Memory Controller  
  - 2 cycles for writes
  - 1 cycle for reads
- Data hazards  
- Structural hazards
